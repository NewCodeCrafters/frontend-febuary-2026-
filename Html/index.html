Introduction to CSS

CSS stands for Cascading Style Sheets. It is a stylesheet language used to describe the presentation of a document written in HTML or XML. CSS defines how elements should be displayed on screen, on paper, or in other media.

What is CSS?

CSS is used to control the layout and appearance of web pages. It allows you to apply styles to HTML elements, such as colors, fonts, spacing, and positioning. With CSS, you can create visually appealing and responsive web designs.

How does CSS work?

CSS works by associating rules with HTML elements. These rules specify how the content of an element should be displayed. A CSS rule consists of a selector and a declaration block. The selector targets the HTML element(s) you want to style, while the declaration block contains one or more declarations that define the styles to be applied.

Example of CSS

Here is a simple example of CSS that changes the background color and text color of a webpage:

            body {
            background-color: lightblue;
            color: darkblue;
            }
